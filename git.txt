git Initial Commit
------------------

Step 1:git init //Initialzied empty git repository

Step 2:git add .

Step 3:git branch -M main

Step 4:git commit -m "Initial Commit" 

Step 5:git remote add origin (git HTTPS URL)

    Incase error
    ------------
    Additional Step: error: remote origin already exists.

    git remote -verbose

    git remote set-url origin (git HTTPS URL)

    ------------x--------------

Step 5:git push -u origin master




Authorized git-User

git config --global user.name "username"
git config --global user.email (useremail)
  
git config --list
